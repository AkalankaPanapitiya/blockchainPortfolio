About

Citizens' identification is more crucial than ever in the digital era, where information is accessible more quickly and easily than ever. The protection and privacy of more conventional forms of identification, such as physical identity cards and passports, are limited. The security and privacy of citizen identity can be enhanced by using a decentralized, immutable system provided by blockchain technology, on the other hand. In the current global environment of growing issues over data privacy and security, the development of a citizen identification system based on blockchain technology is especially crucial. The occurrence of data breaches, identity theft, and other cyberattacks has increased, and conventional identity verification techniques have shown to be vulnerable to these dangers. Additionally, outdated and ineffective citizen identifying systems in many nations contribute to problems like identity fraud and bureaucratic inefficiencies.

Blockchain .

So, we propose a decentralized database system based on Blockchain Technology to Sri Lanka's identity verification and control system to increase the trust, security and integrity of data while avoiding single point of failure and trust in the existing system. Thus, the proposed system will be much more secure and trustworthy, so the government will no longer have to worry about data security and integrity while providing genuine service to citizens in Sri Lanka.

What Blockchain . Do

the research problem we want to solve is Sri Lankan citizen identity information management and verification using a single identity and providing the right to citizens to control the availability of the information which is provided to non-government organizations using private permissioned blockchain technology.

1

Secure
Blockchain System

Provide a permissioned blockchain based citizen identity management and verification system while eliminating the existing weaknesses of current systems.

2

Effecient System

a customized token for the proposed blockchain model which increases throughput of the identity management system based on latest technologies.

3

Having ownership &
control of data

a distributed application to interact with the identity management system and to facilitate citizens to control the availability of their own information based on the latest technologies.

Tools & Technologies

Our team has employed an array of innovative and powerful technologies to develop this impressive blockchain tool. Following are some key tools & technologies

Hyperledger Fabric

Hyperledger Fabric is a permissioned blockchain framework that provides a secure, scalable, and flexible platform for developing enterprise-grade blockchain solutions.

Node Js

A server-side JavaScript runtime environment, useful for building the backend of the system.

Distributed Application

Distributed applications (DApps) are decentralized software applications that run on a peer-to-peer network of computers, rather than a central server.

HTML & CSS

HTML and CSS are markup languages used for building websites and web applications.

React Js

React.js is a JavaScript library for building user interfaces, useful for creating a responsive and intuitive frontend for the system.

GitHub is a web-based platform for version control and collaboration that allows developers to store, share, and manage their code repositories.
